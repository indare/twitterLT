セットアップ手順
１．適当なフォルダで git clone git@github.com:indare/twitterLT.git
２．/twitterLT/twitter4c/setting.phpを編集
２−１．具体的にはde.twitter.comで各種キーを取得
２−２．consumer_key/consumer_secret/access_token/access_token_secret　を書き込む
２−２（補足）．アクセスキーはread writeで取得
２−３．MySqlにつないで下記のスクプトを流して、データベースとテーブル生成
CREATE DATABASE `twitter4c` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci;
USE `twitter4c`;

CREATE TABLE IF NOT EXISTS `tweet_log` (
 `no` int(11) NOT NULL AUTO_INCREMENT,
 `trantime` datetime NOT NULL,
 `tweet` text NOT NULL,
 PRIMARY KEY (`no`)
) ENGINE=InnoDB  DEFAULT CHARSET=utf8;
２−４．MySqlの設定情報（ID・PASS・サーバー名・テーブル名）をsetting.phpに書き込む
３．埋め込みたいディレクトリにbootstrapのセットをUpload
４．埋め込みたいページに下記コードを追記
<script type="text/javascript" charset="utf-8" src="twitter4c/jquery-1.7.1.js"></script>
<script type="text/javascript" charset="utf-8" src="twitter4c/twitter4c.js"></script>
<link rel="stylesheet" href="css/bootstrap.css" type="text/css">
５．あとは表示したい場所に下記html書く。
<div style="margin: 5px;" id="twitter4c"></div>
